### 原理
Timer0
```c
void FreqA()
{
	AUXR &= 0xEF;	//关闭Timer2
	freqValue = (TH2 << 8) + TL2;
	TH2 = 0;
	TL2 = 0;
	AUXR |= 0x10;	//开启Timer2
}

void Timer0_Interrupt() interrupt 1
{
	FreqA();
}
```
